# 后端说明

## 当前后端能力

- 用户注册
- 用户登录
- 用户退出
- 账单保存
- 账单查询
- 预算保存与查询
- 储蓄目标保存与查询
- 仪表盘统计
- 图表数据查询
- 异常记录写入与查询
- 微信账单导入
- 自动同步账单接口骨架

## 启动方式

在 `server` 目录执行：

```bash
npm install
npm start
```

默认服务地址：

```text
http://localhost:3000
```

## 数据库位置

默认数据库文件：

```text
server/data/expense-planner.db
```

也可以通过环境变量 `DB_DIR` 指定新的数据目录。

## 适用阶段

当前方案适合：

- 本地开发
- 课程项目演示
- 个人作品集展示
- 小规模测试

如果后续要做正式线上多人系统，建议升级到 `PostgreSQL`。
